```

NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=template_5otu41z
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=vMcYsQQq4E07XOMkh
```
Ao final, basta rodar o projeto utilizando "npm run dev".
